## 🚀 How to View

No installations needed — these are plain HTML/CSS projects.

**1. Clone the repository**
```bash
git clone https://github.com/datsaryan/InAmigos-Projects.git
cd InAmigos-Projects
```

**2. Open any HTML file in your browser**
```bash
# On macOS
open WebSite/index.html

# On Linux
xdg-open WebSite/index.html

# On Windows
start WebSite/index.html
```

Or simply drag and drop any `.html` file into your browser.
